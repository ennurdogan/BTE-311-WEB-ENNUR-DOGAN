# Hava Durumu Uygulaması - BTE311 Final Ödevi

Bu proje, OpenWeatherMap API kullanarak anlık hava durumu verilerini çeken bir React uygulamasıdır.

## Kullanılan API
- [OpenWeatherMap API](https://openweathermap.org/)

## Özellikler
- Şehir ismine göre arama yapabilme.
- Hava durumuna göre dinamik arka plan değişimi (Güneşli, Yağmurlu, Karlı vs ).
- Yükleniyor ve Hata mesajları veriyor.
- React Hooks (`useState`, `useEffect`) kullanımı.

## Nasıl Çalıştırılır?
1. Projeyi bilgisayarınıza indirin.
2. Terminale `npm install` yazarak paketleri yükleyin.
3. `npm start` komutu ile uygulamayı başlatın.;
